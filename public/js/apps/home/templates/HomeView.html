Welcome

Hey there, thanks for checking out my project.This is a proof-of-concept personal fitness tracking app I built for fun. Due to browser restrictions mainly the inability of backgrounding as well as there being a ton of significantly better solutions already. This would in no way be a practical solution for tracking fitness.

What does it do?

It allows a user to count their steps and track/share there location in realtime.

Want to play around with it?

From a mobile phone (must be a mobile phone or device supporting the motion sensor and geolocation APIs which all modern mobile browsers do), go to the admin page http://websteps.apps.swa.by/admin/. Click the status updater, select track location, and then start tracking steps. You will need to allow the browser to access your location.
